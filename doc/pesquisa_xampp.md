# Pesquisa sobre XAMPP

### Oque é XAMPP?

O XAMPP é um pacote com os principais servidores de código aberto do mercado, incluindo FTP, banco de dados MySQL e Apache com suporte as linguagens PHP e Perl.
Com ele, é possível rodar sistemas como WordPress e Drupal localmente, o que facilita e agiliza o desenvolvimento. Como o conteúdo estará armazenado numa rede local, o acesso aos arquivos é realizado instantaneamente. O pacote de servidores é baixado cerca de 600 mil vezes por mês, de acordo com dados do SourceForge.

### Principais componentes do XAMPP:

O XAMPP, conta com componentes essenciais para o desenvolvimento web. Cada um deles desempenha um papel insubstituível na criação, teste e gerenciamento de aplicações web. 

Abaixo, seguem os principais componentes do XAMPP:

- Apache: Servidor web que processa requisições HTTP e entrega as páginas ao navegador.
- MySQL/MariaDB: Sistema de gerenciamento que armazena e organiza os dados da aplicação.
- PHP: Linguagem que executa os scripts lógicos e gera o conteúdo dinâmico do site.
- phpMyAdmin: Interface gráfica no navegador para gerenciar os bancos de dados visualmente.

### Como realizar a instalação e configuração básica:
Para começar a usar o XAMPP, basta seguir os passos abaixo:

Baixar XAMPP:
- Acesse o site oficial do XAMPP;
- Selecione a versão apropriada para o seu sistema operacional (Windows, Linux, macOS) e faça o download.

Instalar XAMPP:
- Execute o instalador baixado;
- Siga as instruções do assistente de instalação. Durante o processo, você pode escolher os componentes que deseja instalar, como Apache, MySQL, PHP, Perl, entre outros;
- Conclua a instalação e inicie o painel de controle do XAMPP.

Configurar e iniciar os serviços:
- Abra o painel de controle do XAMPP;
- Inicie os módulos Apache e MySQL clicando em “Start” ao lado de cada serviço;
- Verifique se os serviços estão rodando corretamente acessando “http://localhost” no seu navegador. Você deve ver a página inicial do XAMPP, indicando que o servidor está configurado e funcionando.

### Importância do ambiente para desenvolvimento local.
Unindo Apache, MySQL/MariaDB, PHP e Perl em um único pacote, o XAMPP oferece um ambiente de desenvolvimento integrado, que faz com que os desenvolvedores possam trabalhar de maneira coesa, testando e desenvolvendo suas aplicações web em um só lugar.
